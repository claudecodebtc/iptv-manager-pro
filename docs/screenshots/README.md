# Screenshot Plan

Add screenshots to this folder and reference them from the main repository page when available.

Recommended files:

- `main-window.png` - full application window on first launch
- `download-tab.png` - provider URL download workflow
- `edit-tab.png` - group/channel editing workflow
- `url-import-tab.png` - imported URL classification view
- `vlc-preview.png` - playback or mini preview area

Guidelines:

- Use placeholder or non-sensitive playlist data only
- Avoid exposing provider usernames, passwords or private URLs
- Capture at a readable desktop resolution
- Prefer PNG for UI screenshots
